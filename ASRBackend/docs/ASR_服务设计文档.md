## 概述

HearSight ASR（自动语音识别）服务提供音频转文本的功能，支持两种运行模式：本地模式和云端模式。该服务通过统一的 API 接口对外提供服务，屏蔽底层实现差异。

云端模式支持文件上传功能。用户上传的文件会自动上传到 Supabase 存储，然后使用获得的公开 URL 进行云端语音识别。
