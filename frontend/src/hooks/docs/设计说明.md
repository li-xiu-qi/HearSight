# Hooks 设计说明

## 概述

hooks目录包含HearSight前端应用的所有自定义React hooks，提供状态管理和副作用处理的逻辑复用。

## 主要Hooks

### useDataLoader
数据加载和管理hook，负责：
- 转录记录列表获取
- 转录详情加载
- 视频/音频资源管理
- 任务状态同步

### useUrlHandler
URL参数处理hook，功能：
- 页面URL解析
- 参数状态同步
- 路由导航管理

### useVideoSync
视频同步hook，负责：
- 视频播放状态管理
- 时间轴同步
- 字幕定位控制

## 设计原则

1. **单一职责**: 每个hook专注特定功能
2. **组合性**: hooks可灵活组合使用
3. **状态隔离**: 避免不必要的状态耦合
4. **性能优化**: 使用useCallback和useMemo优化重渲染

## 使用模式

- 通过自定义hooks封装复杂逻辑
- 返回状态和操作函数的元组
- 支持TypeScript类型安全
