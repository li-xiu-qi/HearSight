# 工具库设计说明

## 概述

工具库包含HearSight前端应用中的通用工具函数和辅助方法，提供常用的实用功能，减少重复代码编写。

## 主要工具函数

### cn (类名合并函数)
基于 clsx 和 tailwind-merge 实现的类名合并工具，用于处理条件类名和 Tailwind CSS 冲突。

功能特点：
- 支持多个类名参数传入
- 自动处理类名冲突
- 条件性类名应用
- Tailwind CSS 类名优化

使用示例：
```typescript
import { cn } from "@/lib/utils"

// 基本使用
const className = cn("text-red-500", "bg-blue-500")

// 条件使用
const className = cn("text-red-500", isActive && "bg-blue-500")

// 类名冲突处理
const className = cn("p-2 p-4") // 结果为 "p-4"
```

## 设计原则

1. **轻量级**: 工具函数尽可能简单小巧，避免引入大型依赖
2. **通用性**: 提供的功能应该具有广泛适用性
3. **类型安全**: 全面支持 TypeScript 类型检查
4. **无副作用**: 工具函数应该是纯函数，不产生副作用

## 扩展性

可以根据项目需求添加更多通用工具函数，建议按照功能类别组织，保持工具库的整洁和易维护性。
