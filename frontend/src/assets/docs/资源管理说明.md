# 静态资源管理说明

## 概述

assets目录用于存放HearSight前端应用的静态资源文件，包括图标、图片、字体等非代码资源。

## 资源类型

### 图标资源
- SVG图标文件
- 应用Logo
- 第三方图标

### 图片资源
- 装饰性图片
- 示例图片
- 背景图片

### 其他资源
- 字体文件
- 音频文件
- 视频文件

## 使用方式

在组件中引用静态资源：

```typescript
import ReactLogo from '@/assets/react.svg'

// 在JSX中使用
<img src={ReactLogo} alt="React Logo" />
```

或者通过CSS背景图片方式使用：

```css
.background {
  background-image: url('@/assets/background.jpg');
}
```

## 最佳实践

1. **命名规范**: 使用有意义的英文命名，避免使用特殊字符
2. **文件格式**: 优先使用现代Web格式（如WebP图片）
3. **文件大小**: 注意控制资源文件大小，必要时进行压缩优化
4. **版本管理**: 将重要资源纳入版本控制系统

## 优化策略

- 使用适当的图片压缩工具减小文件体积
- 对于大文件考虑CDN分发
- 利用Webpack的资源模块处理静态资源
- 实施懒加载策略提升初始加载速度
