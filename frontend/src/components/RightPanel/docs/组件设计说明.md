# 右侧面板组件设计说明

## 概述

右侧面板是HearSight应用的核心交互区域，提供多标签页式的功能展示，包括转录文本、摘要、聊天问答等功能。

## 组件结构

### RightPanel.tsx

主容器组件，负责：

- 标签页切换逻辑
- 状态管理（当前活跃标签）
- 子组件渲染

### 标签页组件

- **TranscriptTab.tsx**: 转录文本展示
- **SummariesTab.tsx**: 摘要内容展示
- **ChatView.tsx**: 聊天问答界面
- **SegmentsTab.tsx**: 片段管理

### 辅助组件

- **TabToolbar.tsx**: 标签页工具栏
- **LanguageSwitcher.tsx**: 语言切换
- **SearchDialog.tsx**: 搜索对话框
- **TranslateDialog.tsx**: 翻译对话框

## 设计原则

1. **模块化**: 每个标签页独立组件，便于维护和扩展
2. **状态隔离**: 各组件状态独立，避免耦合
3. **响应式**: 支持不同屏幕尺寸的自适应布局
4. **用户友好**: 清晰的视觉层次和交互反馈

## 数据流

- 父组件传递必要props（segments、transcriptId等）
- 子组件通过回调函数与父组件通信
- 使用React hooks管理局部状态

## 扩展性

预留接口支持新增标签页，遵循现有模式即可快速集成。
