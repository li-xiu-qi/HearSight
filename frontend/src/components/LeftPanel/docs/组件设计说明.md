# 左侧面板组件设计说明

## 概述

左侧面板负责展示和管理HearSight应用中的转录记录和处理任务，提供列表视图和状态管理功能。

## 组件结构

### LeftPanel.tsx

主容器组件，负责：

- 标签页布局管理
- 数据传递给子组件
- 响应用户交互事件

### 标签页组件

- **ProcessedTab.tsx**: 已处理转录列表展示和选择
- **TasksTab.tsx**: 任务进度和状态展示

## 设计原则

1. **列表展示**: 使用卡片式布局清晰展示各项内容
2. **状态指示**: 通过颜色和图标直观显示处理状态
3. **交互友好**: 支持点击加载、刷新等操作
4. **性能优化**: 虚拟滚动处理大量数据

## 数据流

- 父组件提供transcripts和jobs数据
- 子组件通过回调函数触发数据更新
- 状态变化通过事件冒泡传递

## 功能特性

- **转录管理**: 展示所有已处理的转录，支持选择和加载
- **任务监控**: 实时显示处理任务的进度和状态
- **状态同步**: 与右侧面板保持数据一致性
